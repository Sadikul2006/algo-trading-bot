- 🤖 Automated Buy & Sell

- 💹 Paper Trading

## 🚀 Installation

Clone the project

```bash
git clone <repository-url>
```

Go to the project folder

```bash
cd algo-trading-bot
```

Create Virtual Environment

```bash
python -m venv venv
```

Activate Virtual Environment

### Windows

```bash
venv\Scripts\activate
```

Install Packages

```bash
pip install -r requirements.txt
```

Run the project

```bash
python main.py
```
